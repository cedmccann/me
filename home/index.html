    A Tutor’s Guide to Calculus

    A calculus-themed website made from scratch with HTML and CSS, the Tutor’s Guide to Calculus was my senior project at Arizona State University. I used to tutor my friends in calculus, so I themed my project off of how I would often explain core calculus principles to my friends. The site consists of a home page and several subpages with each topic, and is responsive, having a desktop, tablet, and mobile version.
